# 🍽️ Restaurant Menu & Order Management System (Part 2)

This project is a Python-based Restaurant Order Management System built using core data structures like lists, dictionaries, and nested combinations.  
It demonstrates real-world operations such as menu handling, cart operations, inventory tracking, and sales analysis.

---

## 📌 Features

### 🔹 Task 1: Explore the Menu
- Displayed menu grouped by categories (Starters, Mains, Desserts)
- Calculated:
  - Total items on menu
  - Available items
  - Most expensive item
  - Items under ₹150

---

### 🔹 Task 2: Cart Operations
- Created a dynamic cart system using list of dictionaries
- Implemented:
  - Add item (with validation)
  - Update quantity (no duplicate entries)
  - Remove item from cart
- Generated final order summary with:
  - Subtotal
  - GST (5%)
  - Total payable amount

---

### 🔹 Task 3: Inventory Tracker with Deep Copy
- Used `copy.deepcopy()` to create a backup of inventory
- Demonstrated independence of original and copied data
- Updated inventory based on cart orders
- Implemented:
  - Stock deduction
  - Low stock warnings
  - Reorder alerts

---

### 🔹 Task 4: Daily Sales Log Analysis
- Calculated total revenue per day
- Identified best-selling day
- Found most ordered item across all days
- Added new sales data dynamically
- Printed all orders using numbered format

---

## 🧠 Key Learnings

- Implementing real-world logic using loops and conditions  
- Data manipulation and analysis
